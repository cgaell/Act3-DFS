# ACTIVIDAD 3 DESARROLLO FULL STACK
__En que consiste?__
Consiste en una API hecha con Node.js en la que maneja un administrador de tareas (TO-DO LIST), tiene 2 tipos de roles, administrador y viewer (usuario normal).

# REQUISITOS
* Tener sistema operativo Windows o MACOS
* Tener instalado Node.js en tu sistema


## Pasos para instalar:
__como la API esta trabajada en localhost, se tienen que realizar los siguientes pasos:__

* 1- Descargar ZIP del repositorio
* 2- Entrar a la terminal
* 3- Ejecutar el siguiente comando:
``` bash
cd Act3-DFS\back
```

habiendo ya entrado a la carpeta de "back" en tu terminal, ejecutar:
``` bash
node server.js
```
Y despues de eso, entras al localhost que te proporciona:
> Servidor corriendo en http://localhost:3000
